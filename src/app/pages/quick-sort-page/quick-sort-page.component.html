Quicksort

Quicksort is a sorting algorithm also known as the divide and conquer algorithm. for example if you send it the following list: 4,3,5,1,2 it will return: 1,2,3,4,5

How does the algorithm work?

This algorithm is also known as the divide and conquer algorithm, since what this algorithm does is divide the list into smaller pieces in each iteration, in order to order the numbers.
For this we will first have to take a number that we will call pivot, this will be the first number of the list, then we will have the left side, in which we will put all the numbers of the list that are less than the pivot and we will also have the right side, where we will put all the numbers greater than the pivot, for example if we have the following list: [5, 4, 7, 1, 3, 2] the pivot would be: 5, and the left would be: [4, 1, 3, 2] , since these numbers are less than 5 (pivot) and the right would be: [7] since seven is the only number in the list greater than 5. Then the process is repeated with each side, both left and right, until Only one number remains, indicating that the list is already sorted.
