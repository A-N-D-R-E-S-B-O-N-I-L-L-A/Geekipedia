Breadth First Seach

Depth First search, as its name says is a search that occurs either in a tree or in a graph at depth to, for example, find if X point connects with another X point.

How does the algorithm work?

The way this algorithm works is that it starts with a random node or the node at the beginning, which we will call the root node, and then it will start to go through all the nodes adjacent to it, entrance to the living room to the kitchen, from the kitchen to the living room and from the living room to the room
